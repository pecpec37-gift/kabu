# 📈 株式投資クイズ道場 - Stock Quiz Master

株取引の知識を楽しみながら学べるブラウザゲームです。  
1ファイル完結・サーバー不要・GitHub Pagesで即公開可能。

## 🎮 ゲームの特徴

- **5ジャンル × 156問以上**のクイズを収録
- **4択問題 ＋ ○×問題** のミックス出題
- **10問ランダム出題**（毎回異なる問題が出る）
- **120秒の制限時間** ＋ 残り時間ボーナス
- ✅ 正解 → +10点 ＋ 時間+5秒
- ❌ 不正解 → 減点なし ＋ 時間−5秒（正解を表示）
- 🏆 結果画面 → 残り時間がボーナス点として加算
- 📚 **間違えた問題を1問ずつ復習表示**
- 📱 スマートフォン対応（レスポンシブデザイン）

| 🌏 世界の市場 | 30問 | NYダウ・NASDAQ・為替・コモディティなど |

## 🚀 GitHub Pages への公開手順

1. このリポジトリをForkまたはClone
2. `index.html` をリポジトリのルートに配置
3. GitHubの「Settings → Pages」からブランチを選択して公開

```
リポジトリ構成:
/
└── index.html    ← これだけでOK！
```

## 💻 ローカルで動かす

HTMLファイルをブラウザで直接開くだけで動作します。  
サーバーや依存関係のインストールは不要です。

```bash
# ダウンロードしてブラウザで開く
open index.html
```

## 🛠 問題の追加・編集

`index.html` の `QUESTIONS` オブジェクトを編集するだけで問題を追加できます。

```javascript
// 4択問題の例
{ type:'mc', q:'質問文', choices:['選択肢A','選択肢B','選択肢C','選択肢D'], answer:0 },
// answer: 正解の選択肢インデックス（0〜3）

// ○×問題の例
{ type:'tf', q:'質問文', answer:true },
// answer: true=正しい / false=違う
```
